## Projekt

Implementierung eines Q-Learning-basierten Encoders für das experimentelle Audioformat **GBABS (Game Boy Audio Bytestream)**.

Das Projekt entstand im Rahmen einer universitären Projektarbeit.

---

## Komponenten

* Q-Learning Trainer
* Teacher Encoder
* Produktivencoder
* GBABS Decoder
* Sparse Q-Table
* HitPot Reward System
* QLTC (Q-Learning Training Container)
